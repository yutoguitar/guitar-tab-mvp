# Guitar Tab MVP — AI Transcription Editor

## このプロジェクト
AIギター転写エディタ。コンテンツ制作のボトルネック（転写が遅い）を解消し、
より多くのリール・Patreonコンテンツを生産可能にする。
Lick Libraryの800アイテムを効率的に拡充するためのツール。

## Tech Stack
Backend: Express 5 + Node.js (body-parser, CORS, multer)
Frontend: Vanilla HTML/JS (index.html, main.js, score-model.js)

## 将来の方向性
- React + TypeScriptへの移行を検討中
- Lick Libraryとの統合（転写 → 直接アップロード）
